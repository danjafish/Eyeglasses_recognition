root_service - скрипт, возврающий пути до фотографий в очках 
data - необходимые данные для запуска модели 
Model - ноутбук и инструкция по получению рабочей модели. 

Предполагается наличие pip

Инструкция по установке:
pip install -r requirements.txt 


Инструкция по запуску:
cd /Eyeglasses_recognition
python -m root_service 'folder_path' (или просто запустить с помощью любого EDA скрипт __main__, указав в переменной path путь до папки)
'folder_path' - полный путь до папки с фотографими, например: /home/fotos/
